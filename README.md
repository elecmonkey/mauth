# MAuth

## Development Prerequisites

- Rust toolchain
- Node.js
- pnpm

Install JavaScript dependencies from the repository root:

```bash
pnpm i
```

then you can run the development stack with:

```bash
pnpm dev
```

If you want to use `bacon run` for the Rust backend, then you can use `pnpm dev:frontend` to boot the frontend dev servers separately.
